## 리액트를 다루는 기술

---

### 1. 리액트 시작

- Yarn 설치

```
# yarn install via npm
npm install --global yarn

# yarn version check
yarn --version
```

- 새 프로젝트 만들어보기

```
$ yarn create react-app hello-react

$ hello-react
$ yarn start
```
